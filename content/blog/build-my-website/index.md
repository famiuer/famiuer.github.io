I bought my domain name in April and initially tried WordPress, then Jekyll on a VPS.
The speed was unstable, so I eventually switched to GitHub Pages.

Why build a website when so many platforms already exist? Because I can. Building it from
scratch is fun, and it helps me learn. It also gives me full control over what I publish.

This website is a place to record my thoughts and ideas.
